## 1. 产品概述

SafuSkill 是一个基于 **BNB Chain** 的 AI 技能代币化平台，允许开发者和创作者将自己的 AI Agent 技能与一枚 **BEP-20 代币** 绑定，通过 **Flap Protocol** 完成链上发射。

**核心价值主张：**

- 将 AI 技能资产化：每个 AI 技能对应唯一一枚代币，形成可交易的链上凭证
- 公平启动保护：支持初始购买以抵御狙击机器人攻击
- 透明费率：1% 协议手续费由智能合约永久锁定，分配比例公开不可篡改
- 开发者激励：Skill 奖励池持续为技能创作者提供收入来源

**适用对象：**

| 角色 | 使用场景 |
|------|---------|
| AI 开发者 | 为自己的 Agent 创建对应代币，实现技能价值的链上流通 |
| 技能创作者 | 上传技能文件或导入 GitHub 仓库，参与生态奖励分配 |
| 投资者 / 社区用户 | 在技能市场中发现、购买并持有 AI 技能代币 |

## 2. 核心概念

### 2.1 AI 技能（Skill）

AI 技能是平台的基本资产单元。每一个技能代表一个可运行的 AI Agent 能力模块，支持以下形式上传：

- **ZIP 压缩包**：包含 Agent 代码、配置文件、依赖描述
- **GitHub 仓库**：直接链接至公开仓库，系统自动从 `SKILL.md` 或 `README` 中提取元信息

### 2.2 BEP-20 代币

每枚代币通过 Flap Protocol 在 BNB Chain 上部署为标准 BEP-20 合约。代币具有：

- **最大供应量**：10 亿枚
- **初始可购买上限**：8 亿枚（预留 20% 作为协议储备）
- **股票代码（Ticker）**：由创建者自定义，全大写字母，唯一标识

### 2.3 Flap Protocol

Flap Protocol 是 SafuSkill 底层使用的代币发射协议，负责：

- BEP-20 合约的标准化部署
- 初始流动性建立
- 费率分配逻辑的链上执行

## 3. 功能模块详解

### 3.1 导航与全局操作

顶部固定导航栏提供以下入口：

| 导航项 | 说明 |
|--------|------|
| 技能市场（Skill Market）| 浏览和购买已上线的 AI 技能代币 |
| 发射台（Launchpad）| 创建并发射新的技能代币（当前页面）|
| 文档（Docs）| 开发者文档和集成指南 |
| 开发者（Developer）| API 接口、SDK 和开发资源 |

**右侧操作区：**

- **语言切换**：支持中文 / English 一键切换，全页面文字实时同步
- **通知铃**：链上事件和系统通知
- **钱包地址**：显示当前已连接的 Web3 钱包（绿色圆点表示在线状态）

---

### 3.2 发射代币表单

发射台页面核心区域为"代币信息"表单卡片，包含以下字段：

#### 代币图片（Token Image）

- 位置：表单左侧正方形区域
- 支持格式：PNG、JPEG、WEBP、GIF
- 大小限制：5MB
- 交互：点击区域触发文件选择，选择后即时预览；边框在悬停和已上传状态有视觉反馈

#### 代币名称（Token Name）*必填*

- 代币的完整名称，将在链上和平台内展示
- 示例：`Code Review Agent Token`
- 建议：清晰反映技能功能，长度不超过 50 字符

#### 股票代码（Ticker）*必填*

- 代币的交易符号，通常为 2-6 位大写字母
- 示例：`CRVW`
- 用途：在交易界面、价格图表和创建流程中作为主要标识符
- 注意：股票代码将动态显示在后续所有 Modal 和通知文案中；留空时系统以 `JIOJIO` 作为占位展示

#### 描述（Description）*选填*

- 对代币用途和所绑定技能的文字说明
- 多行文本框，支持换行输入
- 建议包含：技能的核心能力、适用场景、与其他代币的差异化价值

---

### 3.3 选择技能

每枚代币必须与一个已有或新上传的 AI 技能绑定，此为**必填项**。

**操作方式一：搜索已有技能**

在搜索框内输入关键词，系统在平台已收录的 AI 技能库中进行模糊匹配，选中后与本次代币关联。

**操作方式二：上传新技能**

点击"上传绑定技能"区域，弹出[上传技能 Modal](#34-上传技能-modal)，完成新技能的上传和审核后自动关联至本次代币。

---

### 3.4 上传技能 Modal

点击"上传绑定技能"区域触发，提供两种上传方式：

#### Tab 1：上传文件

| 字段 | 必填 | 说明 |
|------|------|------|
| 技能名称 | 是 | 技能的显示名称，显示于技能市场列表 |
| 描述 | 否 | 技能功能的详细说明 |
| 技能文件 | 是 | `.zip` 格式压缩包，最大 50MB |

上传流程：
1. 选择 `.zip` 文件（点击或拖放至上传区）
2. 点击「上传 & 扫描」
3. 文件经 **GoPlus AgentGuard** 自动安全扫描
4. 扫描通过后技能入库，可与代币关联

#### Tab 2：从 GitHub 导入

| 字段 | 必填 | 说明 |
|------|------|------|
| GitHub 仓库链接 | 是 | 完整仓库 URL，需为公开仓库 |
| 自定义技能名称 | 否 | 留空则自动读取仓库名称或 `SKILL.md` 中的 name 字段 |
| 自定义描述 | 否 | 留空则自动读取 `SKILL.md` 前置信息或 `README` 内容 |

> **自动提取优先级：** SKILL.md frontmatter → README 首段 → 仓库描述（About）

点击「导入 & 扫描」后，系统克隆仓库并经 GoPlus AgentGuard 扫描，通过后技能入库。

### 3.5 创建代币 Modal

填写完表单所有必填项后，点击「创建代币」按钮触发此 Modal。

#### 初始购买设置

为保护代币免受**狙击机器人**（Sniper Bot）在发射瞬间的大量买入攻击，创建者可选择在部署时同步购买一定数量的代币：

- **购买上限**：8 亿枚（总供应量的 80%）
- **支付单位**：BNB（余额实时显示于输入框上方）
- **换算提示**：输入 BNB 数量后，界面实时显示预计可获得的代币数量
- **超额退款**：若实际所需 BNB 少于输入金额，差额自动退回钱包
- **初始购买为可选项**，输入 0 或留空则跳过

#### 钱包余额显示

Modal 顶部以绿色在线指示灯实时展示当前钱包的 BNB 余额，帮助用户合理规划购买金额。

#### 部署成本

每次代币合约部署需支付约 **0.001 BNB** 的链上 Gas 费用（不包含初始购买金额）。

---

### 3.6 链上部署流程

点击「立马创建！」后，系统进入三阶段自动化部署流程，全程通过右下角 **Toast 通知**实时反馈进度：

```
阶段一：钱包授权
  └─ 等待用户在钱包插件中确认签名
  └─ 按钮显示旋转 Loading + "等待钱包授权中..."
  └─ 授权成功后按钮短暂变绿确认

阶段二：上传 IPFS（约 2.5 秒）
  Toast: "正在启动 {TICKER} ..."
  └─ 代币元数据（名称、描述、图片、关联技能）上传至 IPFS
  └─ 获取不可变的 IPFS CID 作为元数据地址

阶段三：部署合约（约 3 秒）
  Toast: "合约部署中..."
  └─ Flap Protocol 在 BNB Chain 上部署 BEP-20 合约
  └─ 合约地址写入元数据，完成链上绑定

完成
  Toast: "🎉 {TICKER} 代币已成功发射！"
  └─ 「查看代币详情」按钮跳转至代币详情页
  └─ Toast 在 5 秒后自动淡出消失
```

## 4. 费率机制

每笔 **{TICKER}** 代币链上交易将收取 **1%** 协议手续费，由智能合约自动执行分配，无需任何人工干预。

### 4.1 分配比例

| 接收方 | 比例 | 用途 |
|--------|------|------|
| 平台（SafuSkill）| 30% | 平台运营、产品迭代与生态维护 |
| 开发者奖励 | 40% | 分配给代币创建者及技能开发者 |
| Skill 奖励池 | 30% | 激励生态内活跃技能的持续维护与创新 |

> 平台内「交易费率说明」卡片中可展开查看带进度条的可视化分配图。

### 4.2 费率锁定机制

- 上述 1% 费率和三方分配比例**均由部署时的智能合约代码固化**
- 合约部署后任何人（包括 SafuSkill 团队）均无法单方面修改
- 持币者可通过区块链浏览器（BscScan）验证合约源码

## 5. 安全机制

### 5.1 GoPlus AgentGuard 扫描

所有通过「上传技能」功能提交的技能文件和 GitHub 仓库，在入库前均经过 **GoPlus AgentGuard** 自动安全扫描，检测包括但不限于：

- 恶意代码注入
- 权限越界调用
- 已知漏洞模式匹配
- 数据泄露风险

扫描未通过的技能将被拒绝入库，不可与代币关联。

### 5.2 防狙击保护

创建代币时的初始购买机制允许创建者预先持仓，从而：

- 拉高初始价格基础，提高狙击机器人的攻击成本
- 稳定代币发射初期的价格曲线
- 建立创建者与代币的长期利益绑定

### 5.3 钱包安全

- 平台不托管任何私钥
- 所有链上操作均需用户在本地钱包（MetaMask 等）中手动确认签名
- 超额 BNB 退款逻辑写入合约，平台无法截留

## 7. 字段规范

| 字段 | 必填 | 类型 | 限制 | 说明 |
|------|------|------|------|------|
| 代币图片 | 否 | 图片文件 | PNG/JPEG/WEBP/GIF，≤5MB | 展示于代币详情页和市场列表 |
| 代币名称 | 是 | 文本 | 建议 ≤50 字符 | 链上元数据的 name 字段 |
| 股票代码 | 是 | 文本 | 2-6 位，建议大写字母 | 合约 symbol 字段，部署后不可更改 |
| 描述 | 否 | 多行文本 | 建议 ≤500 字符 | 链上元数据的 description 字段 |
| 选择技能 | 是 | 关联选择 | 必须关联已通过审核的技能 | 合约元数据中的技能绑定字段 |
| 网站 | 否 | URL | 以 http/https 开头 | 代币官网链接 |
| 推特 | 否 | 文本 | @ 用户名或完整 URL | 项目社交媒体账号 |
| 初始购买 | 否 | 数字 | 0 ~ 对应 8 亿枚代币的 BNB 数量 | 发射时同步买入的 BNB 数量 |

## 8. 术语表

| 术语 | 解释 |
|------|------|
| **BEP-20** | BNB Chain 上的代币标准，等同于以太坊的 ERC-20 |
| **Flap Protocol** | SafuSkill 使用的代币发射与流动性协议 |
| **Ticker / 股票代码** | 代币的交易简称，如 BTC、ETH |
| **IPFS** | 星际文件系统，用于存储代币元数据的去中心化存储网络 |
| **GoPlus AgentGuard** | 专为 AI Agent 技能设计的链上安全扫描服务 |
| **狙击机器人（Sniper Bot）** | 在代币发射瞬间自动抢购大量代币的套利程序 |
| **Gas 费** | BNB Chain 上执行智能合约操作所需支付的网络手续费 |
| **CID** | IPFS 内容标识符，用于唯一定位链上存储的元数据 |
| **BscScan** | BNB Chain 的区块链浏览器，可查验合约源码和交易记录 |
